# Roman Pashto Dialect Classification

This project focuses on low-resource natural language processing by studying dialect variation within Roman Pashto language across multiple geographic regions.

Roman Pashto lacks standardized writing conventions, making dialect classification a challenging problem. The research investigates whether multilingual transformer models can learn regional linguistic variation despite inconsistent spelling patterns and informal writing structures.

The project is being developed as research-oriented work with the long-term goal of contributing to underrepresented language technology.

## Research Focus

* Dialect classification across regional Pashto variations
* Transformer fine-tuning for low-resource languages
* Linguistic variation analysis
* Roman Pashto dataset creation and preprocessing

## Technologies Used

* Hugging Face Transformers
* XLM-RoBERTa
* RoBERTa
* PyTorch
* Python

## Goal

To explore how modern NLP systems can improve language understanding for low-resource regional languages while contributing toward publishable research.
